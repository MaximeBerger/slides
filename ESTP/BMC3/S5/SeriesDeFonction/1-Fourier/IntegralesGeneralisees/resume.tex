 Les intégrales impropres suivantes sont-elles convergentes?
 \begin{multicols}{3}
 \begin{enumerate}
    \item $\int_0^1(\ln t) d t$
    \item $\int_0^1 \frac{d t}{(1-t) \sqrt{t}}$
    \item $\int_0^{+\infty} e^{-\sqrt{t}} d t$
    \item $\int_0^{+\infty}(\ln t) e^{-t} d t$
    \item $\int_0^{+\infty} t(\sin t) e^{-t} d t$
 \end{enumerate}
 \end{multicols}
 Les intégrales impropres suivantes sont-elles convergentes?
 \begin{multicols}{3}
 \begin{enumerate}
    \item $\int_0^{+\infty} \frac{d t}{e^t-1}$
    \item $\int_0^{+\infty} \frac{t e^{-\sqrt{t}}}{1+t^2} d t$
    \item $\int_0^1 \cos ^2\left(\frac{1}{t}\right) d t$
 \end{enumerate}
 \end{multicols}

 Discuter, suivant la valeur du paramètre $\alpha \in \mathbb{R}$, la convergence des intégrales impropres suivantes :
 \begin{multicols}{3}
 \begin{enumerate}
    \item $\int_0^{+\infty} \frac{d t}{t^\alpha}$
    \item $\int_0^{+\infty} \frac{e^{-t}-1}{t^\alpha} d t$
    \item $\int_0^{+\infty} \frac{t-\sin t}{t^\alpha} d t$
    \item $\int_0^{+\infty} \frac{\arctan t}{t^\alpha} d t$
 \end{enumerate}
 \end{multicols}
