Comment lancer l'application interactive :

1. Ouvrez simplement le fichier "index.html" avec votre navigateur web préféré (Chrome, Firefox, Edge, Safari).
2. Naviguez entre les onglets pour voir les différents exercices.

Aucune installation n'est requise. Une connexion internet est nécessaire uniquement pour charger les bibliothèques graphiques (Plotly, Tailwind, MathJax) au premier chargement.
