Structure du Calcul Vectoriel

Positionnement Relatif et Courbes de Bézier

Avantages du positionnement relatif :

  • Facilité de modification : Changer le centre modifie toute la structure
  • Calculs automatiques : Positions calculées par angle et distance
  • Symétrie parfaite : Distribution équilibrée autour du centre

Courbes de Bézier :

  • Quadratiques (Q) : Un point de contrôle, courbes simples
  • Cubiques (C) : Deux points de contrôle, courbes plus fluides
  • Paramètre de courbure : Contrôle l'intensité de la courbe

Opérations Vectorielles

Produit Scalaire

$$\vec{u} \cdot \vec{v} = |\vec{u}| |\vec{v}| \cos(\theta)$$

Mesure l'angle entre deux vecteurs

Produit Vectoriel

$$\vec{u} \times \vec{v} = |\vec{u}| |\vec{v}| \sin(\theta) \vec{n}$$

Vecteur perpendiculaire aux deux vecteurs

Gradient et Applications

Gradient d'un champ scalaire

$$\nabla f = \left(\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z}\right)$$

Applications en physique

  • Champ électrique : $\vec{E} = -\nabla V$
  • Force conservative : $\vec{F} = -\nabla U$

Exemple d'Implémentation

Positionnement relatif :

// Définir le centre
const centerX = 400, centerY = 250;

// Calculer position par angle et distance
function getPosition(angle, distance) {
    const x = centerX + distance * Math.cos(angle * Math.PI / 180);
    const y = centerY + distance * Math.sin(angle * Math.PI / 180);
    return { x: Math.round(x), y: Math.round(y) };
}

// Utilisation
const pos = getPosition(45, 150); // 45°, distance 150px
// pos.x = 506, pos.y = 356

Courbes de Bézier :

// Courbe quadratique simple
function getBezierPath(startX, startY, endX, endY, curvature = 0.3) {
    const midX = (startX + endX) / 2;
    const midY = (startY + endY) / 2;
    const controlX = midX + curvature * (endY - startY);
    const controlY = midY - curvature * (endX - startX);
    return `M ${startX} ${startY} Q ${controlX} ${controlY} ${endX} ${endY}`;
}

// Utilisation dans SVG
<path d="M 400 250 Q 300 200 200 150" stroke="red" fill="none"/>
