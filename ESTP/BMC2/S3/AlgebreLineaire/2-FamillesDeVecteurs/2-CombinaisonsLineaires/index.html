Algèbre linéaire

2 — Combinaisons linéaires

Un fil conducteur : « synthétiser un objet à partir d’ingrédients »

À la fin de cette séquence, vous serez capables de :

  • Décider si \(v\in\mathrm{Vect}(b_1,\dots,b_p)\) à l’aide d’un système \(B\lambda=v\).
  • Calculer explicitement des coefficients \(\lambda_i\) d’une combinaison linéaire.

Prérequis : coordonnées dans une base ; mettre en équation une combinaison.

Menu

    Problématique / Accroche

    Peut-on synthétiser un signal ou une couleur à partir d’ingrédients donnés ?

    Idée : un mélange = une combinaison linéaire.

    Définitions

    • Combinaison linéaire : \(v=\sum_{i=1}^p \lambda_i b_i\).
    • Sous-espace engendré : \(\mathrm{Vect}(b_1,\dots,b_p)=\{\sum_i \lambda_i b_i\}\).
    • Test d’appartenance : \(v\in \mathrm{Vect}(b_1,\dots,b_p)\iff \exists\,\lambda: B\lambda=v\), avec \(B=(b_1\;\cdots\;b_p)\).
    • Outil : élimination de Gauss pour résoudre \(B\lambda=v\).

    Schéma global / Visualisation

    Selon \(p\) et les dépendances linéaires :

    • \(p=1\) et \(b_1\neq 0\) : une droite par l’origine.
    • \(p=2\) indépendants dans \(\mathbb R^3\) : un plan par l’origine.
    • \(p\geq n\) avec rang \(n\) : tout l’espace \(\mathbb R^n\).

    Exemple clé (\n R^3)

    \(b_1=(1,0,1),\; b_2=(0,1,1),\; b_3=(1,1,0)\). Décider si \((2,1,3)\in\mathrm{Vect}(b_1,b_2,b_3)\).

    On pose \(B=\begin{pmatrix}1&0&1\\0&1&1\\1&1&0\end{pmatrix}\), \(v=\begin{pmatrix}2\\1\\3\end{pmatrix}\), et on résout \(B\lambda=v\).

    Voir la démo interactive « Gauss pas à pas » ci-après.

    Exemple clé (\n \mathcal P_2)

    Écrire \(X\) comme combinaison de \((1, X-1, X^2)\).

    Chercher \(\alpha,\beta,\gamma\) tels que \(\alpha\cdot 1 + \beta(X-1) + \gamma X^2 = X\). On compare les coefficients.

    Activité : réinterpréter un système

    Résoudre \(xu_1 + yu_2 = b\) = trouver une combinaison de \(u_1,u_2\) donnant \(b\).

    • Aucun \((x,y)\) : \(b\) est hors du plan engendré.
    • Une infinité : \(b\) est dans le plan et les solutions forment une droite d’inconnues.
    • Unique : \(u_1,u_2\) indépendants et \(b\) dans \(\mathrm{Vect}(u_1,u_2)\).

    Pièges fréquents

    • Confondre combinaison linéaire et produit scalaire : \(\sum\lambda_i b_i\) ≠ \(\langle u,v\rangle\).
    • Oublier la fermeture : le résultat doit rester dans l’ensemble considéré.
    • Penser que toute partie de \(\mathbb R^n\) est un sous-espace : il faut contenir \(0\) et être stable par combinaisons.
    • Confondre dépendance et engendrement : engendrer un espace n’implique pas l’indépendance.
