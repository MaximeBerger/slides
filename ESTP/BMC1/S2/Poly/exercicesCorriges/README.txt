ORGANISATION DES CORRIGÉS
==========================

Ce dossier contient les corrigés des exercices du polycopié.

STRUCTURE :
-----------
- Chaque chapitre a son propre sous-dossier (ex: R-Integrales/)
- Chaque exercice a son propre fichier .tex (ex: exo-2.1.tex pour l'exercice 1 du chapitre 2)
- Les fichiers PDF sont générés automatiquement

NOMMAGE :
---------
- exo-X.Y.tex où X = numéro de chapitre, Y = numéro d'exercice
- Exemple : exo-2.3.tex = Exercice 3 du Chapitre 2 (Intégrales)

COMPILATION :
-------------
Utiliser le script !make-corrige.bat pour compiler un corrigé individuel.

URL D'HÉBERGEMENT :
-------------------
Les PDF sont hébergés sur : https://votre-domaine.fr/corriges/BMC1/S2/
(À configurer dans configPoly.tex, variable \urlcorriges)
